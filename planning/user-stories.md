# User Stories

---

- As a store owner, I want to create and update my menu online so that my customers always see accurate offerings.

- As a customer, I want to browse the store’s menu online so that I can view available items before placing an order.

- As a customer, I want to select menu items and submit an order online so that I can order conveniently from my device.

- As a store owner, I want to receive new orders in a dedicated business dashboard so that I can efficiently manage incoming orders.

- As a store owner, I want to confirm or decline orders in the dashboard so that customers are quickly updated on their order status.

- As a customer, I want to receive notifications when my order is confirmed or declined so that I know if my order will be fulfilled.

- As a store owner, I want to view a list of current and past orders so I can manage my business activity.

- As a customer, I want to view the status of my submitted orders so I know what to expect.

- As a store owner, I want to edit or remove menu items as needed so my online menu is always up to date.

- As a customer, I want a simple, clear ordering experience so that I can place my order without confusion.

- As a store owner, I want to log in securely to access my dashboard so that my business information is protected.

- As a customer, I want to place orders without creating an account so that I can order quickly and easily (optional, if supported).
